Muuta fonttikokkoa Javascriptillä

Huom! Kaikki muutokset tapahtuvat Javascriptin avulla.
Listan fonttikokoa muutetaan alla olevaa painonappia painamalla. Listan oletusfonttikoko on 16px.

Tämän listan fonttikoko muuttuu napist painamalla

  1. Ensimmäinen napin painallus muuttaa fontikooksi 20px
  2. Toinen napin painallus muuttaa fontikooksi 24px
  3. Kolmas napin painallus muuttaa fontikooksi 28px
  4. Neljäs napin painallus muuttaa fontikooksi 32px
  5. Viides napin painallus palauttaa fonttikoon 16px
